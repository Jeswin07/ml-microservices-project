            Common "Failed to fetch" causes:


            1. Port-forward not running — run both:

              kubectl port-forward svc/auth-service 8004:8004

              kubectl port-forward svc/api-gateway 8000:8000


            2. CORS not enabled on your FastAPI services.

              Add CORSMiddleware with allow_origins=["*"]


            3. Wrong endpoint path — adjust URLs in Settings above.


            4. HTTP vs HTTPS mismatch — keep everything on HTTP locally.
